## Scripts from other mods

By default, the language server works automatically with everything in your workspace, the base game, and Community Patch - Shared Imports. To pull in `.ws` files from **another mod**, add the directories here.

> `!` If the mod you want to add **replaces** base game scripts, check out the next step instead.

[Add Script Folder](command:witcherscript.addAdditionalScriptDirectory)
