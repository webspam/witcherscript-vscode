## Legacy scripts

For older mods that use **edited copies** of base game scripts (e.g. `scripts/game/gameplay/r4Game.ws`) instead of using `@` annotations.

> `!` A script in this list will be used instead of the base game script of the same name.

If you open a script that's replacing a base games cript, a small status bar icon is shown in the bottom right.

Add the mod's `scripts/` folder - the one containing `core/`, `engine/`, and/or `game/`:

```
📁 scripts
    📁 core
    📁 engine
    📁 game
        📁 gameplay
```
