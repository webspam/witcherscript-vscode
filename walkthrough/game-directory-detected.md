## Game directory detected

Your Witcher 3 install was detected automatically and the language server
is using it to resolve base game scripts. Nothing to do here.

To set the path manually, 
[open WitcherScript settings](command:workbench.action.openSettings?%22%40ext%3Awebspam.witcherscript-language-features%20gameDirectory%22)
and check `gameDirectory`. A manually set value always wins over auto-detection.
